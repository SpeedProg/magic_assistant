Creating and Populating a Deck

Create a new Deck

First create a deck in the Card Navigator.
  • Open a Card Navigator view, by clicking on the header.
  • Find Deck folder and right click on it.
  • Select New...-> Deck from the context menu.
  • Type a name in the Name field. Click Finish.
    • Deck will be automatically open.

    Populate a deck from printed cards

    Now to populate a deck open MTG Database view. If your deck is made from limited number of sets, it is better to select them in the filter, to simplify the search. To do that click on Filter button on the view header (Or select Filter... from view or context menu), switch to Set Filter on the left and select one or more sets.

    Now type Ctrl-F or select Find... from the view menu (triangle button in the view header). Search panel will be open.

    Bring up Card Info view to see results of your search.

    Type name of card in the search field, you don't have to type whole name, search would show up cards as you type. As soon as you see your card in Card Info view press "+" button. Card will be added to your deck. If card name consists of several words, you can type only first letters of each word in capital letters, for example to find "Shield of the Righteous" type SOTR in the search field. If it is not your card you can keep typing until full name is typed or press Enter to cycle through search results.

    If your cards were sorted by name already you can either type + several times to achieve needed number, or double click on card in the deck and edit count of cards directly. You also can edit count in the Count cell itself.

    Populate a deck using text import

    If you have a deck contents in electronic form you import it using Deck import. Select a new empty deck and select Import... from a context menu (right click menu). Select a file or clipboard option and press Next. You will see a preview of your deck import. Click Finish to finish the importing. Format of "Deck Classic" (sample): 
    Card Name x4
Another Card Name (Set) x 3
    or 
    4 x Card Name 
3 x Another Card Name
