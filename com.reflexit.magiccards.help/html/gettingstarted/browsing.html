Browsing Cards

This is application for browsing Magic The Gathering cards (see official website).

To browse Magic cards switch to Magic perspective, from Window->Open Perspective. If you running standalone version this is only perspective available.

Magic perspective contains 2 view: Card Description and Magic Database. Magic Database show all available cards (loaded in local database). By default it will only show one instance of a card with the same name for the latest edition.

Sorting

You can sort by any field by clicking on column header. If you click again sorting order is reversed.

Filtering

You can filter cards in the view by opening filter. You can select one of more criteria from the filters. Items that belong to the same group (such as Color) would be selected using logical OR. Item that belong to different groups would be selected using logical AND.

For example and you select color Blue and White and Type Creature and Enchantment, you will get all creates and all enchantments that are Blue or White (including multicolors).
