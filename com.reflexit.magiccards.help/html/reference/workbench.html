Workbench

Concepts

Application workbench consists from several views that show different types of information, and interact with each other, for example when object is selected in one view the other one can show more details on that object.

You can change layout of your views at any time by dragging them around by the header.

Magic Assistance contains the following views:

  • MTG Database View - lists all available cards
  • My Cards View - lists cards that belong to one or more of your collections.
  • Deck View - lists cards in a deck
  • Card Info - shows card details of selected card
  • Card Navigator - shows list of user decks and collections organized in folders
