Commands

Common commands for card list views

Filter... Opens a Filter dialog Context menu and toolbar button on all card list views
Reset Filter Clears all filtering on cards lists Context menu and toolbar button on all card list views
Preferences... Opens a preferences dialog which allows to adjust columns Context menu and toolbar button on all card list views
(View Menu) Opens a view menu All views (button located on a view toolbar)

Generic commands

  • Delete [DEL] - deletes an element(s)
  • Copy [CTRL+C] - copy element(s) to clipboard or text representation of the element in text clipboard
  • Paste [CTRL+V] - paste element(s) from clipboard
  • Find [CTRL+F] - opens a find control

View specific commands

  • Increase [+] - increase count of the cards for selected card piles
  • Decrease [-] - decrease count of the cards for selected card piles
  • Move to > - menu to move card to selected collection/deck. Collection must be opened to show up in this menu.
  • Split pile... - split selected card (card pile) into 2 piles (if count is more than 1)
  • Edit Card... - edit card properties, note: card count and some other fields can be edited right in the cell
  • Add to > - menu to add card to a selected collection/deck. Collection must be opened to show up in this menu.
  • Add to Default Collection [INSERT] - Adds card to "main" collection
  • Add to Active Collection [+] - Adds card from db to active deck or collection (active the most recently open or focused)
  • Show Other Sets - for selected card show if this is printed in other sets, opens the Printings view
  • Show All Instances - for selected card show how many instances of this card you own, and in which deck/collections, opens the Printings view
