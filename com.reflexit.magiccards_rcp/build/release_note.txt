Notes
* Requires java at least 1.8 to run
* Has to be unzipped before running using any unzip program, windows XP zipped folder copy action does not work properly
* For Linux 32 use specific binary distribution (different than linux default)
* Ubuntu 13.10+ requires additional installation of libwebkit
